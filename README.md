# 🎮 Catch the Box Game

A fun and simple browser game where you try to catch a moving box and increase your score!

## 🚀 How to Play

* Click on the red box 🟥
* Each click increases your score 📈
* The box moves every second, so be quick!

## 🛠️ Built With

* HTML
* CSS
* JavaScript
